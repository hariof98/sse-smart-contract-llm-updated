# examples/ — Small runnable demos

Tiny scripts that illustrate the data shapes and APIs. Not tests, not part of a
run — just sanity checks you can read and execute.

## Files

| File | Shows |
|---|---|
| `demo_schema.py` | What a `GroundTruth` and a tool `Prediction` look like in code, including a false-positive example. |

## Run

```bash
python3 examples/demo_schema.py
```
